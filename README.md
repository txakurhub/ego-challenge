## Cómo instalarlo y correr el proyecto localmente en tres simples pasos

### 1. Clonar el proyecto

```

$ git clone https://github.com/txakurhub/ego-challenge.git

```

### 2. Instalar dependencias

```

$ npm install

```

### 3. Levantar el proyecto

```

$ npm run dev

```
